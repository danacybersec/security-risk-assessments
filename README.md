## Methodology

Each assessment in this portfolio follows a consistent methodology:

**Risk Score = Likelihood (1–5) × Impact (1–5)**

| Score | Risk Level | Response Timeline |
|-------|------------|-------------------|
| 15–25 | 🔴 High | Immediate (30 days) |
| 8–14  | 🟡 Medium | Short-term (90 days) |
| 1–7   | 🟢 Low | Next review cycle |

**Assessment process for each engagement:**
1. Organization and technology environment review
2. Threat and vulnerability identification
3. Likelihood and impact scoring
4. Risk register development
5. Detailed findings with HIPAA regulatory citations
6. Prioritized remediation roadmap

## HIPAA Regulatory Alignment

All assessments map findings to applicable HIPAA Security Rule provisions, including:

- `45 CFR §164.308(a)(1)` — Security Management Process / Risk Analysis
- `45 CFR §164.308(a)(5)` — Security Awareness and Training
- `45 CFR §164.312(a)(1)` — Access Control / Unique User Identification
- `45 CFR §164.312(a)(2)(iv)` — Encryption and Decryption
- `45 CFR §164.312(b)` — Audit Controls
- `45 CFR §164.308(b)(1)` — Business Associate Contracts
- `45 CFR §164.310(a)(1)` — Facility Access Controls
